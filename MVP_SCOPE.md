# Burch Platform MVP Scope

## Goal

Launch the first public version of Burch Platform with the minimum feature set required to onboard customers and partners.

---

## Customer Features

- Create Account
- Login
- Browse Events
- Browse Hotels
- Browse Restaurants
- Search Experiences
- Book an Event
- Reserve a Hotel
- Reserve a Restaurant
- Secure Payments
- View Booking History

---

## Partner Features

- Register Business
- Create Listings
- Manage Availability
- View Bookings
- Basic Analytics

---

## Admin Features

- Approve Partners
- Manage Users
- Manage Listings
- View Platform Statistics

---

## Not Included in MVP

- Mobile Apps
- Multi-language Support
- Loyalty Program
- Public API
- Advanced AI Analytics

These will be introduced in future releases.
